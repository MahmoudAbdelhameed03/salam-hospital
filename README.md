#  Hospital Management System – Frontend Project

##  Project Overview
This project is a **Hospital Management System (Frontend)** built using HTML, CSS, Bootstrap, and JavaScript.  
The website allows users to explore services, view doctors, and book appointments through an interactive user interface.

---

##  Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  

---

##  Project Pages
- Home Page  
- Doctors Page  
- Departments / Services Page  
- Doctor Details Page  
- Book Appointment Page  
- Contact Page  
- Login Page  
- Register Page  

---

##  Features
- Responsive multi-page layout  
- Reusable components  
- Clean UI design  
- Form validation using JavaScript  

---

##  How to Run
1. Download or clone the project.
2. Open `index.html` in your browser.
